# AI Chat Assistant

A ChatGPT-style AI chat interface built with vanilla HTML, CSS, and JavaScript.

## Features

- ChatGPT-style conversational UI
- Streaming responses with typewriter effect
- Markdown rendering in messages
- Code blocks with syntax highlighting and copy button
- Conversation history management
- Suggestion cards for quick prompts
- Fully responsive design for mobile and desktop

## Tech Stack

- **HTML5** — Semantic markup and structure
- **CSS3** — Custom styling with CSS variables, flexbox, and grid
- **Vanilla JS** — No frameworks or dependencies

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ezechias1/ai-chat.git
   cd ai-chat
   ```

2. Open `index.html` in your browser, or serve with any static file server:
   ```bash
   npx serve .
   ```

No build step required.
